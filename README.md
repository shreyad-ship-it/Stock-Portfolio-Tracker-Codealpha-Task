# Stock Portfolio Tracker using Python

## Project Overview
The Stock Portfolio Tracker is a Python-based application that helps users calculate the total value of their stock investments. Users enter the quantity of stocks they own, and the program computes the overall investment value using predefined stock prices.

This project demonstrates the use of Python dictionaries, loops, arithmetic operations, user input handling, and file handling.

## Features
- Stores stock prices using a dictionary.
- Accepts stock quantity input from the user.
- Calculates the investment value for each stock.
- Computes the total portfolio value.
- Saves the result to a text file.
- Simple and easy-to-use interface.

## How to Run
1. Open the project in Google Colab or any Python IDE.
2. Run the Python script.
3. Enter the quantity for each stock when prompted.
4. View the total investment value.
5. The result will be saved in a file named `portfolio.txt`.
